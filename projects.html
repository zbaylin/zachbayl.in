Rolltrax More info
I co-founded Rolltrax alongside Ohad Rau and Dr. Brian Patterson in 2017. Our original goal was to create a localized cloud based school attendance system for students who participate in work-based learning programs. This concept quickly ballooned into a full software stack, including a website, an API, a desktop app, and a mobile app. Now Rolltrax helps coordinators of work-based learning programs manage attendance, employee evaluations, routine job visit planning, payroll information, and more. BookWorm More info
BookWorm is a project I developed for the FBLA Coding and Programming event in 2019. The project description mandated that submissions create a GUI that helps manage the distribution of eBooks from librarians to patrons. I chose to write my GUI in Python using the Qt GUI framework, combining the two using the officially supported Qt for Python bindings. It supported numerous features, like manager/user stratification, a centralized cloud based backend, PDF DRM generation, and more. It won first place at the Georgia FBLA State Leadership Conference, and fifth place at the National Leadership Conference in San Antonio, TX.
biblotech More info
bibliotech is an application my team and I designed for the FBLA Mobile Application Development event in 2018. The main goal of the project was to provide librarians and students an easy way to navigate a physical library. The app includes more advanced features, like an interactive map of the library, as well as integration with other services like Google Books and GoodReads. We wrote the mobile app in Flutter, and used Ruby + Sinatra on the backend. Our submission one fourth place at the Georgia FBLA State Leadership Conference.
Traveller More info
Traveller was an extra credit project a partner and I developed for my MATH 3012 (Applied Combinatorics) class at Georgia Tech. It was a proof-of-concept application that used OpenStreetMap routing data to solve specific cases of the Travelling Salesman problem, using both the greedy algorithm and Dijkstra's algorithm. The program is written in Nim, with data being inputted via CSV.
SECurity-py More info
SECurity-py originally started as a project to fetch quarterly SEC filings from hedge funds and analyze trends between quarters. It ended up expanding to other types of filings, and now supports Excel output, and can function as an Excel macro itself. My father initially requested this project from me, and it eventually became a fully functional application
rofi-wifi-menu More info
rofi-wifi-menu is a plugin for the rofi alt-space launcher that helps people connect to WiFi networks on NetworkManager-based systems using nmcli. It has amassed over 100 starts on Github, and is widely used in the Linux/Unix community on systems with basic desktop environments, such as openbox.
